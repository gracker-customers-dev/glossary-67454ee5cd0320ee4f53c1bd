UICC

Universal Integrated Circuit Card.

Description

The Universal Integrated Circuit Card (UICC) is a smart card that is used in mobile devices to securely store subscriber information and facilitate communication between the device and mobile networks. It includes the functionalities of both SIM (Subscriber Identity Module) cards and USIM (Universal Subscriber Identity Module) cards, allowing for enhanced services and security. UICCs are essential for enabling mobile telephony and data services, as they authenticate the user's device on the network and manage the connection to the service provider. Beyond telecommunications, UICCs can also store applications that allow for mobile payment systems, secure access, and identity verification. The UICC is designed to comply with global standards set by organizations such as the 3rd Generation Partnership Project (3GPP) and the International Telecommunication Union (ITU), ensuring compatibility across different devices and networks. As mobile technology evolves, UICCs continue to adapt, incorporating advanced security features and supporting various mobile network technologies including 2G, 3G, 4G, and 5G.

Examples

  • A UICC used in a smartphone for mobile network access.
  • A UICC that supports mobile payment applications.

Additional Information

  • UICCs can store multiple profiles for different network operators.
  • They can be physically embedded in devices or used as removable cards.
